üìã Gmail App Password Setup Instructions

If emails are not being delivered to Gmail, you need to configure App Password authentication:

  1. Go to Google Account Security
  2. Enable 2-Step Verification if not already enabled
  3. Click on "App passwords"
  4. Select "Mail" and "Other (Custom name)"
  5. Enter Django FTC App as the name
  6. Copy the 16-character password generated
  7. Update FTC/settings.py:
EMAIL_HOST_PASSWORD = 'your-16-character-app-password'  # Replace with generated password
EMAIL_BACKEND = 'django.core.mail.backends.smtp.EmailBackend'  # Switch back to SMTP
